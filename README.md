# AI Agentic Platform 🚀
A collection of projects for learning and building high-performance AI agents using LangChain, LangGraph, and LangSmith.

## 📁 Projects

### 🔬 [AI Research Assistant](./ai-research-assistant)
A stateful, autonomous research agent powered by **Groq** and **LangGraph**.
- **Features**: Web Search, Math Tools, RAG, and a Streamlit Dashboard.
- **Stack**: Python, LangChain, Groq, Tavily, Streamlit.

### ☁️ [AWS Multi-Agent Assistant](./aws-infra-assistant)
A multi-specialist agent system for AWS infrastructure management using a Supervisor/Router pattern.
- **Features**: Specialized agents (EC2, RDS, Lambda), Multi-turn state persistence, AWS CLI integration.
- **Stack**: Python, LangChain, LangGraph, AWS CLI.
